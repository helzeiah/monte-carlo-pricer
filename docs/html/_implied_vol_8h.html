Solves for implied volatility under the Black-Scholes model.

Uses a Newton-Raphson root-finding (with bisection fallback) to find the volatility σ such that BS_price(option, σ) = targetPrice.

Parameters
optionthe Option parameters (S, K, T, r, q, type).
targetPricethe market price to invert.
tolthe convergence tolerance (default: 1e-8).
maxIterthe maximum number of Newton iterations (default: 50).
Returns
the implied volatility σ.
Exceptions
std::runtime_errorif convergence fails within maxIter.
